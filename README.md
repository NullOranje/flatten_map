# FlattenMap

A tool for flattening nested maps.

## Installation

The package can be installed by adding `flatten_map` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:flatten_map, "~> 0.1.0"}
  ]
end
```
